# حاسبني Pro — نظام ERP المتكامل

## 🚀 تشغيل البرنامج

### المتطلبات
- **Node.js** النسخة 18 أو أحدث من [nodejs.org](https://nodejs.org) (اختر LTS)

### للمرة الأولى
1. فك ضغط الملف
2. انقر مرتين على **`تثبيت-وتشغيل.bat`**

### للمرات التالية
- انقر مرتين على **`تشغيل.bat`**

---

## 🔑 بيانات الدخول

| المستخدم | الاسم | كلمة المرور | الدور |
|---|---|---|---|
| 👑 مدير النظام | admin | 010999 | Admin |
| 💼 كاشير | cashier | 010666 | Cashier |
| 📦 مدير المخزن | store | 010123 | Store |
| 📊 محاسب | account | 010321 | Accountant |

## ✅ الأقسام المتاحة

| القسم | الوصف |
|---|---|
| 🏠 الرئيسية | لوحة تحكم مع إحصائيات وتنبيهات |
| 💰 نقطة البيع (F2) | إنشاء فواتير البيع |
| 📦 المنتجات | إدارة المخزون والأسعار |
| 🛒 المشتريات | فواتير الشراء |
| 📋 المبيعات | سجل الفواتير |
| 👥 العملاء | إدارة العملاء والديون |
| 🏭 الموردون | إدارة الموردين |
| 🔄 المرتجعات | مرتجعات البيع والشراء |
| 💳 الخزنة | الحركات المالية |
| 📊 التقارير | تقارير شاملة |
| ⚖️ الأرصدة | كشوف حساب العملاء والموردين |
| 🤝 الشركاء | إدارة الشركاء وتوزيع الأرباح |
| 🏭 الإنتاج | أوامر الإنتاج والخامات |
| 💹 المالية | لوحة التحكم المالية |
| 👤 المستخدمين | إدارة الحسابات والصلاحيات |

## ⌨️ اختصارات لوحة المفاتيح

| المفتاح | الوظيفة |
|---|---|
| F2 | فتح نقطة البيع |
| F3 | البحث في نقطة البيع |
| F9 | إتمام البيع |
| F10 | مسح السلة |
| Escape | إغلاق النافذة |
| + | زيادة الكمية |
| - | تقليل الكمية |
