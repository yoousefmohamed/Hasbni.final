# 🚀 طريقة تشغيل حاسبني v9

## الطريقة الصحيحة خطوة بخطوة

### 1️⃣ تثبيت المتطلبات (مرة واحدة فقط)
افتح CMD داخل مجلد hassibni_v9 واكتب:
```
npm install
```
انتظر حتى يكتمل التحميل (قد يأخذ دقيقة أو دقيقتين)

### 2️⃣ تشغيل البرنامج
```
npm start
```

---

## ⚠️ أخطاء شائعة وحلولها

| الخطأ | السبب | الحل |
|-------|-------|------|
| `Missing script: "build"` | كتبت `npm run build` بالخطأ | اكتب `npm start` |
| `Cannot find module 'electron'` | لم تثبّت المتطلبات | شغّل `npm install` أولاً |
| `electron is not recognized` | Electron لم يتثبت | شغّل `npm install` من جديد |
